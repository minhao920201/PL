### Week3
*list[ ]* : 可有次序的放入不同類型的資料

    增加 : .append() / .insert() / .extend()
    刪除 : del[] / .remove() / .pop()
*tuple( )* : 元素不可被修改，且與list相似有順序

*set{ }* : 無序、元素不可重複、不可為list

    增加 : .add() / .update()
    刪除 : .remove() / .pop() / .clear()
*dict{ key:value }* : 無序、無索引、以key和value方式儲存，其中key不可重複
